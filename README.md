# GameToken WIP

This repository includes:

1. GameToken Main contract
2. GameToken AirDrop contract
3. GameToken Game contract

This project requires Solidity ^0.4.4.

Install dependencies:
    ``npm i -g solc``
    ``npm i -g truffle``
    ``npm i -g ethereumjs-testrpc``
    ``npm install``

Run tests:
    ``truffle test``
